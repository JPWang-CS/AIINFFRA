# Code Review Skill

## 用途
审查 CUDA/Triton 算子代码，检查正确性、性能和最佳实践。

## 审查维度

### 1. 正确性
- 内存访问边界检查
- 线程块/网格配置验证
- 同步点位置正确性
- 数据类型匹配

### 2. 性能
- Memory coalescing
- Bank conflict 识别
- Occupancy 评估
- Shared memory 利用率

### 3. 最佳实践
- 错误处理
- 可读性
- 命名规范
- 注释质量

## 输出格式
```
🔍 代码审查报告

✅ 正确性: [通过/问题]
- [具体发现]

⚡ 性能: [评分/10]
- [优化建议]

💡 最佳实践: [评分/10]
- [改进建议]
```

## 调用时机
- 用户完成算子初版时
- 性能不达标需要诊断时
- 请求"帮我看看代码"时
