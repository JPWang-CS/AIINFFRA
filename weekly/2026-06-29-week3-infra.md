## 卡点 / 怎么解决的

1. **Agent 职责边界模糊**：之前 ascend-to-gpu-coach 一个 Agent 包揽所有（导航 + 教学 + 审查），越来越臃肿。**解法**：拆成三层——base-coach 做导航和状态管理，ascend-to-gpu-coach 做 GPU 学习和理论，cuda-tutor 做算子级深钻。每个 Agent 职责清晰，通过 base-coach 协调调用。

2. **跨对话记忆丢失**：每次新对话 Agent 都要重新了解用户背景和进度，浪费 token。**解法**：建立 Memory 系统（`.claude/memory/` + `agent-memory/`），把用户画像、项目进度、反馈偏好持久化，Agent 启动时自动加载。
